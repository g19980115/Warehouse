网络导入中，目标内容为标准的json，需按照以下字段进行编写

字段释义：

name 导入后的名称；subname 导入后的副名称。
title 外部展示的标题，用于描述此规则。
icon 图标链接。
type 类型, 0 活动/1 小程序/2 链接/3 Shell。
pkg 当type为0/2时，此项为包名；type为1时，此项为小程序id。
path 路径、shell指令。
andVer 需要的最低安卓api版本。
author 作者。
workMode 工作模式，0 普通，1 Root，2 Shizuku。
